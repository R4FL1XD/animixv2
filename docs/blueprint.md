# **App Name**: Animix

## Core Features:

- Home Screen: Display the latest anime releases, top anime, and anime movies fetched from https://www.sankavollerei.com/anime/samehadaku/home.
- Anime Details: Display detailed information about each anime, including the title, poster, episodes, and release date.
- Genre Filtering: Allow users to filter anime by genre, based on genre lists retrieved from the API.
- Search: Users will be able to find an anime using keyword

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to reflect the depth and immersion of anime.
- Background color: Light Gray (#F0F0F0) to ensure content is easily readable and visually accessible.
- Accent color: Vibrant Pink (#E91E63) for highlighting interactive elements and calls to action.
- Body font: 'Inter', a grotesque-style sans-serif font, will be used to ensure a modern, machined, objective, and neutral look. Headline font: 'Space Grotesk', a proportional sans-serif font with a computerized, techy, and scientific feel, will be used for a clear, engaging presentation.
- Use simple, outline-style icons for navigation and actions.
- Implement a grid-based layout for displaying anime posters and information.
- Use subtle transition animations when loading content or navigating between pages.
